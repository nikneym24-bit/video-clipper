# Slicr

Автоматический конвейер видеоклипов: от Telegram-канала до VK Клипов.

## Быстрый старт (macOS)

```bash
# Двойной клик по scripts/dev.command или:
./scripts/dev.command

# Или вручную:
pip install -e .
python -m slicr
```
